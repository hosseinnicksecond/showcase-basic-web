Web Demo Using IMDb Data

A demo web application showing various features such as:

 - Javalin web framework
 - Embedded Jetty web server
 - Thymeleaf templating engine
 - Jdbi relational data access
 - Hikari database connection pooling
 - Hibernate Validator data validation
 - Gson JSON serialization/deserialization
 - jQuery UI user interface widgets
 - Datatables tabular data display
 - Skeleton responsive UI/layout boilerplate

It is not intended to be a fully functioning web application. It implements basic select and update functionality, but has no create or delete capabilities. The site uses publicly available data from IMDb to demonstrate its features.
